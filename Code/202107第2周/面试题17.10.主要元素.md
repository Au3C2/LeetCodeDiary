# [面试题 17.10. 主要元素](https://leetcode-cn.com/problems/find-majority-element-lcci/)



数组中占比超过一半的元素称之为主要元素。给你一个 **整数** 数组，找出其中的主要元素。若没有，返回 `-1` 。请设计时间复杂度为 `O(N)` 、空间复杂度为 `O(1)` 的解决方案。

 

**示例 1：**

```
输入：[1,2,5,9,5,9,5,5,5]
输出：5
```

**示例 2：**

```
输入：[3,2]
输出：-1
```

**示例 3：**

```
输入：[2,2,1,1,1,2,2]
输出：2
```

# 代码

## 1.  摩尔投票

```python
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        candidate = -1
        count = 0
        for num in nums:
            if count == 0:
                candidate = num
            if num == candidate:
                count += 1
            else:
                count -= 1
        count = 0
        for num in nums:
            if num == candidate:
                count += 1
        return candidate if count > len(nums)/2 else -1
```



## 2. 哈希计数



```python
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        counter = Counter(nums)
        most_num = counter.most_common(1)[0][0]
        return most_num if counter[most_num] > len(nums)/2 else -1
```
