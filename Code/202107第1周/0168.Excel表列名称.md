# [168. Excel表列名称](https://leetcode-cn.com/problems/excel-sheet-column-title/)

难度简单387收藏分享切换为英文接收动态反馈

给定一个正整数，返回它在 Excel 表中相对应的列名称。

例如，

```
    1 -> A
    2 -> B
    3 -> C
    ...
    26 -> Z
    27 -> AA
    28 -> AB 
    ...
```

**示例 1:**

```
输入: 1
输出: "A"
```

**示例 2:**

```
输入: 28
输出: "AB"
```

**示例 3:**

```
输入: 701
输出: "ZY"
```

数学 简单 

# 代码

要减1

```python
class Solution:
    def convertToTitle(self, n: int) -> str:
        s = ''
        while(n):
            n -= 1
            s = chr(n%26+65) + s
            n = n//26
        return s
```
